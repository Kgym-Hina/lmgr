Logo Manager

Herzlich willkommen beim Logo Manager. Bitte lest erst aufmerksam die folgenden Informationen durch um eine Fehlbedienung und eventuelle Beschädigung eures Gerätes zu vermeiden!

1. Allgemeines

Logo Manager ist ein einfach zu bedienendes Tool um die Startbildschirme (logo.img) bestimmter Xiaomi Smartphones anzuzeigen, anzupassen und zu installieren.

Es basiert auf Qt und ist somit auf verschiedenen Betriebssystemen wie Windows, Linux und MacOS einsetzbar. Alle erforderlichen Dateien sind enthalten, sodass keine Installation erforderlich ist.

2. Anzeigen von Logos

Zum Anzeigen des Inhaltes eines logo.img kann dieses über "Öffnen" geladen werden.

Es werden nun die enthaltenen Logos angezeigt. Über jedem Logo stehen zusätzliche Details wie Auflösung, Position und Größe.

3. Anpassen von Logos

Als Basis muss immer eine originale Datei für das Ziel-Gerät verwendet werden!

Diese bekommt man z.B. aus der Firmware-Datei von https://xiaomifirmwareupdater.com/firmware für das gewünschte Gerät.

3.1 Importieren

Über "Importieren" kann das jeweilige Logo durch ein Eigenes ersetzt werden. Dieses sollte möglichst die selbe Größe wie das Original haben, wird aber bei Bedarf auch entsprechend skaliert.

Nachdem alles angepasst wurde kann das neue Image über "Speichern" erzeugt werden.

3.2 Exportieren

Über "Exportieren" kann das jeweilige Logo extrahiert werden. Dabei werden die Formate BMP, JPG und PNG unterstützt.

Weiterhin können die Rohdaten als RAW gespeichert werden.

4. Installieren von Logos

Logos können auch direkt auf das Gerät installiert werden. Dies geschieht ausdrücklich auf eigene Gefahr und kann das Gerät beschädigen!

Dazu muss der Bootloader entsperrt und Android-Debugging in den Entwickleroptionen aktiviert sein.

Sind diese Vorraussetzungen erfüllt kann das Gerät per USB-Kabel mit dem PC verbunden werden.

Nun wird über "Flashen" das gewünschte Image ausgewählt. Danach öffnet sich der Installationsdialog und es wird versucht das verbundene Gerät zu erkennen.

War dies erfolgreich wird das Gerät im Fastboot-Modus neu gestartet. Dies dauert ca. 5-10 Sekunden, also Geduld!

Anschließend wird das Image in die Logo-Partition installiert und das Gerät wieder ins System gestartet. Das neue Logo sollte dabei nun angezeigt werden.

4.1 Probleme

Problem : es wird kein Gerät erkannt
Lösung : unter Windows kann es erforderlich sein die benötigten Treiber für adb/fastboot zu installieren

Problem : es erscheint "unauthorized device"
Lösung : auf dem Gerät die angezeigte Anfrage bestätigen

Problem : es erscheint "no permissions"
Lösung : auf dem Gerät den USB-Modus von Aufladen auf Übertragen ändern
